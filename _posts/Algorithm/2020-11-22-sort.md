### 冒泡排序

从头遍历到尾, 遇到相邻的元素逆序就交换, 重复n次

```cpp
// bubble sort

#include <bits/stdc++.h>

using namespace std;

/* 
 * Time: 
 * n * (n - 1)
 * => n^2 - n
 * => O(n^2)
 */
void bubble_sort(vector<int>& vals)
{
    for (size_t i = 0; i < vals.size(); ++i)
    {
        for (size_t j = 0; j < vals.size() - 1; ++j)
        {
            if (vals[j] > vals[j + 1])
                swap(vals[j], vals[j + 1]);
        }
    }
}
```

### 选择排序

从未排序的序列中选择最小的元素跟首元素交换, 重复上述过程到结束

```cpp
// Selection Sort

#include <bits/stdc++.h>

using namespace std;

/*
 * Time : 
 * n + n - 1 + n - 2 + n - 3 + ... + 2 + 1
 * => (n + 1) * (n - 1) / 2
 * => n^2 / 2 - 0.5
 * => O(n^2)
 */

void selection_sort(vector<int>& vals)
{
    for (size_t i = 0; i < vals.size(); ++i)
    {
        size_t min_idx = i;
        int min_val = vals[min_idx];
        for (size_t j = i + 1; j < vals.size(); ++j)
        {
            if (vals[j] < min_val)
            {
                min_idx = j;
                min_val = vals[j];
            }
        }
        swap(vals[i], vals[min_idx]);
    }
};
```
